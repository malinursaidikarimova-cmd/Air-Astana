# AI-use disclosure

| Вопрос | Ответ |
|---|---|
| Какие ИИ-инструменты использовались? | Codex / ChatGPT. |
| Для чего использовался ИИ? | Для помощи в структуре проекта, подготовке веб-приложения, создании Excel/CSV input, оформлении README, методологии и disclosure. |
| Какие части кода были подготовлены с помощью ИИ? | HTML/CSS/JavaScript веб-приложения, вспомогательные скрипты для переноса данных из Excel, документация проекта. |
| Какие данные проверялись вручную? | Строки баланса, ОПиУ, ОДДС, формулы input, сходимость баланса, CFO, CFI, CFF, Capex, FCF. |
| Где ИИ мог ошибиться? | В интерпретации строк отчетности, сопоставимости классификаций между годами, выборе proxy-показателей и формулировке аналитических выводов. |
| Что остается ответственностью автора проекта? | Финальная проверка источников, подтверждение цифр отчетности, выбор assumptions прогноза и итоговая аналитическая позиция. |

ИИ использовался как технический помощник. Ответственность за финальную версию проекта, проверку данных и выводы остается за автором работы.
