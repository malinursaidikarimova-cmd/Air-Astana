# Air Astana Financial Statement Analysis

Веб-приложение для проекта по анализу финансовой отчетности Air Astana.

Проект реализует цепочку из ТЗ:

`Отчетность -> Качество -> Анализ -> Прогноз -> Вывод`

## Что входит в проект

- `app/` - веб-приложение на HTML, CSS и JavaScript.
- `data/инпутАФО.xlsx` - Excel input workbook с листами `Баланс`, `ОПиУ`, `ОДДС`, `Коэффициенты`, `Инпут`, `Проверки`, `Источники`.
- `data/air_astana_sample.csv` - CSV input, который приложение загружает по умолчанию.
- `methodology/methodology_air_astana.md` - методология анализа.
- `AI_USE_DISCLOSURE.md` - раскрытие использования ИИ.
- `package.json` - команда для локального запуска статического приложения.

## Данные

Период анализа: 2021-2025.

Валюта и масштаб: KZT, тыс. тенге.

Источник данных: финансовая отчетность Air Astana, собранная в Excel input. CSV создан из листа `Инпут` и используется приложением как машинно-читаемый файл данных.

## Как запустить

Вариант 1: открыть файл напрямую:

```text
app/index.html
```

Вариант 2: запустить локальный сервер:

```bash
python -m http.server 8000
```

Открыть:

```text
http://localhost:8000/app/
```

Вариант 3, если используется npm:

```bash
npm start
```

## Блоки приложения

1. **Отчетность / Input**  
   Показывает нормализованные строки отчетности: выручку, прибыль, активы, обязательства, капитал, CFO, CFI, CFF, Capex.

2. **Качество**  
   Проверяет полноту input, сходимость баланса, качество прибыли через CFO / чистая прибыль, FCF и динамику ключевых строк.

3. **Анализ**  
   Показывает динамику выручки, прибыли, денежных потоков, коэффициенты и отраслевые метрики.

4. **Прогноз**  
   Строит base и stress сценарии на 3 года с изменяемыми assumptions.

5. **Вывод**  
   Формирует итоговое аналитическое резюме без инвестиционной рекомендации.

## Как обновить input

1. Обновить Excel `data/инпутАФО.xlsx`.
2. Экспортировать лист `Инпут` в CSV с такой же структурой, как `data/air_astana_sample.csv`.
3. Заменить `data/air_astana_sample.csv`.
4. При необходимости заменить встроенный `sampleCsv` в `app/app.js`.

## Ограничения

- Проект является учебным analytical tool, а не инвестиционной рекомендацией.
- Прогноз является сценарным и зависит от assumptions.
- Debt proxy считается как `loans + lease liabilities`.
- Capex берется из ОДДС как приобретение основных средств.
